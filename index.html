The Many Federal Lands of the American West

This map was built using HTML, CSS, JavaScript, and D3.js. It was especially challenging due to using a large data set while also ensuring the the map remained responsive.

It shows the US federal government owned lands across the 11 western states. The purpose of the map is allow the user to discover these tracts, the percentage by state of Federal owned land, and the agency responsible for their management. The proportional circles can be displayed or hidden and by hoverign on the circle view state level information. By hovering over the many tracts, the user will see the agency, tract name, and tract type.

  • See the map: The Many Federal Lands of the American West
  • Data Sources: USGS National Map and The Congressional Research Service's report Federal Land Ownership: Overview and Data

Kenya Livestock Populations

This map was built using HTML, CSS, and Mabbox.js on a Mapbox base map.

It shows livestock populations in Kenya. By hovering over the rings the user is provided the density of livestock per county and population information on cattle, sheep, and goats.

  • See the map: Kenya Livestock Populations
  • Data Source: Kenya Open Data
